# Анализ графа вызовов с помощью Phasar

## Описание

Модуль анализа графа вызовов (`CallGraph`) позволяет построить и проанализировать граф вызовов функций в LLVM IR коде с использованием библиотеки Phasar.

## Возможности

1. **Построение графа вызовов** - анализирует все вызовы функций в программе
2. **Статистика вызовов**:
   - Общее количество функций
   - Количество мест вызовов
   - Прямые и косвенные вызовы
   - Внешние функции
   - Рекурсивные функции

3. **Анализ цепочек вызовов**:
   - Функции-листья (не вызывают другие функции)
   - Функции верхнего уровня (точки входа)
   - Обнаружение рекурсии

4. **Экспорт в формат DOT** для визуализации с помощью Graphviz

## Использование

### Базовый запуск

```bash
./build/PhASAR-advisor <llvm_ir_file> callgraph
```

## Визуализация

Для визуализации графа вызовов используется Graphviz:

```bash
# PNG
dot -Tpng callgraph.dot -o callgraph.png

# SVG (масштабируемый формат)
dot -Tsvg callgraph.dot -o callgraph.svg

# PDF
dot -Tpdf callgraph.dot -o callgraph.pdf
```

## Примечания

1. **Intrinsic функции** (llvm.dbg.*, llvm.lifetime.*) автоматически пропускаются
2. **Внешние функции** отмечаются в выводе и на графе
3. **Рекурсия** обнаруживается только прямая (функция вызывает саму себя)
